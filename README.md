# 👁️ Face Blurring Project

This project automatically detects and blurs unknown faces in images using the `face_recognition` and `OpenCV` libraries in Python.  
It’s useful for privacy protection in photos or datasets where only specific people should be visible.

---

## 🚀 Features
- Detects all faces in input images.
- Matches faces against a known dataset.
- Blurs unrecognized faces.
- Saves processed images automatically.
